# EndoBrain: The Clinical Knowledge Compiler\n\nInspired by Karpathy's 'LLM Knowledge Bases', EndoBrain is an autonomous wiki that transforms medical raw data into a structured Obsidian vault.\n\n## Core Pillars\n1. **Data Ingest**: Raw PDFs stored in `raw/`.\n2. **The Compiler**: LLM-driven synthesis of atomic markdown notes with backlinks.\n3. **Front-end**: Obsidian IDE for clinical exploration.\n4. **Evolving Logic**: Incremental updates when new trials/research are added.\n\n## Current Status\n- Initializing structure...\n- Pending Williams Textbook ingestion.
